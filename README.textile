This is a project developed during a Yahoo! internal hackday.

It is an iPad app for Yahoo! Meme, a light blogging platform from Yahoo! that is currently available in several languages including English, Portuguese, Spanish, Bahasa and Traditional Chinese.

This app has the objective to offer a native looking way to navigate and consume content from user's dashboard on Meme.

Development powered by Titanium Mobile framework that allows development for iOS platform using Javascript instead of ObjC.

h2. Requirements

- iOS 4.1 SDK
- Titanium Mobile SDK 1.4.1.1

h2. Features

- oAuth Authentication
- Navigate on Meme's dashboard
- Repost and Comment from within the App
